# 🎤 Out Of The Woods Media

## 📌 Description

A frontend web project inspired by Taylor Swift’s musical eras, featuring albums, tour highlights, and fan engagement sections. It showcases responsive design, structured navigation, and visually themed UI using HTML and CSS, built for demonstrating core web development and UI/UX concepts in an academic setting.

---

## ✨ Features

* 🎶 Album showcase section
* 🌍 "Eras Tour" highlights display
* 👥 Community join section
* 🛍️ Navigation bar with multiple pages (Home, Albums, Tour, Shop)
* 💻 Responsive and visually styled UI
* 🎨 Themed design inspired by music eras

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Basic Web Design Principles
